# ChatCar 개인정보 처리방침

**시행일**: 2026년 5월 11일
**최종 개정일**: 2026년 6월 16일

ChatCar(이하 "회사" 또는 "서비스")는 「개인정보 보호법」 등 관련 법령상의 개인정보보호 규정을 준수하며, 이용자의 개인정보를 안전하게 보호하기 위해 최선을 다하고 있습니다. 본 개인정보 처리방침은 회사가 제공하는 ChatCar 모바일 애플리케이션(이하 "앱") 및 관련 서비스에 적용됩니다.

---

## 1. 수집하는 개인정보 항목

### 1-1. 회원가입 및 본인 식별

| 항목 | 수집 방식 | 비고 |
|------|----------|------|
| 이메일 주소 | 사용자 직접 입력 또는 카카오 계정 | 회원 식별, 알림 발송 |
| 닉네임 | 카카오 계정 (선택 동의) | 앱 내 사용자명 표시 |
| 카카오 사용자 식별자 (user_id) | 카카오 OAuth 연동 | 회원 식별 |
| 비밀번호 | 사용자 직접 입력 (이메일 가입 시) | 암호화 저장 |

### 1-2. 차량 진단 서비스 이용

| 항목 | 수집 방식 | 비고 |
|------|----------|------|
| 차량 정보 (제조사/모델/연식) | 사용자 입력 또는 OBD 자동 인식 | 진단 서비스 제공 |
| 차대번호(VIN) | OBD 자동 인식 또는 사용자 입력 | 차량 식별 |
| 차량 번호판 | 사용자 입력 | 차량 식별 |
| OBD 센서 데이터 | OBD 어댑터 연결 시 자동 수집 | 실시간 진단, AI 분석 |
| AI 진단 질문/응답 | 사용자 입력 + AI 생성 | 진단 이력 관리 |

### 1-3. 정비 이력 관리

| 항목 | 수집 방식 | 비고 |
|------|----------|------|
| 영수증 사진 | 사용자 촬영/업로드 | OCR 처리 후 즉시 삭제 |
| 정비 항목 및 금액 | OCR 추출 + 사용자 수정 | 정비 이력 저장 |
| 정비소 정보 | 영수증에서 추출 | 정비 이력 저장 |

### 1-4. 자동 수집 정보

| 항목 | 수집 방식 | 비고 |
|------|----------|------|
| 기기 정보 (OS, 앱 버전) | 자동 수집 | 호환성, 오류 분석 |
| 앱 이용 로그 | 자동 수집 | 서비스 개선 |
| IP 주소 | 자동 수집 | 보안, 이상 접근 감지 |

### 1-5. 결제 관련 정보 (유료 플랜 이용 시)
결제 정보는 PG(결제 대행사)를 통해 처리되며, 회사는 결제 내역(결제일, 결제 금액, 상품명)만 보관합니다. 카드 번호, CVC 등 결제 수단 정보는 회사가 직접 수집하지 않습니다.

## 2. 개인정보의 수집 및 이용 목적

회사는 수집한 개인정보를 다음 목적으로 이용합니다.

1. **회원 관리**: 회원 식별, 본인 확인, 회원 가입 의사 확인, 부정 이용 방지
2. **차량 진단 서비스 제공**: OBD 데이터 분석, AI 진단, 차량 건강 점수 산출
3. **정비 이력 관리**: 영수증 OCR, 정비 항목 분류 및 저장, 이력 조회
4. **사용량 관리**: 무료/유료 플랜 한도 관리, 결제 처리
5. **서비스 개선**: 이용 패턴 분석, 신규 기능 개발
6. **고객 지원**: 문의 응대, 공지사항 발송
7. **법적 의무 이행**: 관련 법령 준수, 분쟁 해결

## 13. 탈퇴 후 정보 보존 (어뷰징 방지 및 재가입 시 데이터 복원)

회사는 회원 탈퇴 시 모든 개인정보를 삭제하는 것을 원칙으로 합니다.
다만, **무료 체험 어뷰징 방지 및 재가입 시 데이터 복원(서비스 연속성)**을 위해
다음 정보를 **6개월간** 보존하며, 보존 기간 경과 시 자동 파기합니다.
(본 보존 및 복원에 대한 동의는 회원가입 시 본 처리방침 동의로 갈음합니다.)

### 보존 항목

| 항목 | 보존 형태 | 용도 |
|------|----------|------|
| 카카오 사용자 식별자 | SHA-256 해시 | 재가입 시 동일 사용자 식별 |
| 이메일 주소 | SHA-256 해시 (정규화 후) | 재가입 시 동일 사용자 식별 |
| 마지막 구독 플랜·만료일·사용량 | 평문 | 재가입 시 플랜·사용량 복원 |
| 차량 정보 (제조사·모델·연식·차대번호·주행거리 등) | 평문 (개인정보) | 재가입 시 차량 정보 복원 |
| 진단 이력 | 평문 (개인정보) | 재가입 시 진단 이력 복원 |

### 보존 목적

- **어뷰징 방지**: 동일 사용자가 탈퇴 후 재가입하여 무료 체험을 반복 사용하는 것을 차단
- **데이터 복원**: 재가입 시 이전 구독 상태·사용량·차량 정보·진단 이력 복원

### 처리 방식

- 식별 정보(카카오 ID, 이메일)는 **SHA-256 해시**로 변환되어 저장됩니다(원본 복호화 불가).
- 복원 대상인 **차량 정보·진단 이력**은 복원을 위해 평문(개인정보)으로 보존되며, **보존 목적(복원·어뷰징 방지) 외의 용도로는 이용하지 않습니다.**

### 보존 기간 및 파기

- **6개월**: 보존 기간 경과 후 자동 삭제(파기)됩니다.
- 보존 기간 내라도 **본인 확인 후 즉시 완전 삭제 요청**이 가능하며, 개인정보 보호책임자에게 문의해 주시기 바랍니다.
