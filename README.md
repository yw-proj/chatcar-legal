# ChatCar 법적 문서 호스팅

ChatCar 앱의 개인정보 처리방침 및 이용약관을 GitHub Pages로 호스팅합니다.

> ⚠️ **저작권 안내**: 본 저장소의 모든 문서는 ChatCar 서비스 운영을 위한 법적 문서입니다.
> 무단 복제 및 자기 서비스 약관으로의 사용을 금지합니다. © 2026 ChatCar.

## 접근 URL

- 메인: https://yw-proj.github.io/chatcar-legal/
- 개인정보 처리방침: https://yw-proj.github.io/chatcar-legal/privacy/
- 이용약관: https://yw-proj.github.io/chatcar-legal/terms/

## 용도

- 앱 스토어 등록 시 필수 URL (Google Play / Apple App Store)
- 카카오 비즈앱 검수 시 제출 자료
- 앱 사용자가 외부 브라우저로 열람 가능

## 파일 구조

```
chatcar-legal/
├── _config.yml       # Jekyll 설정
├── index.md          # 랜딩 페이지
├── privacy.md        # 개인정보 처리방침
├── terms.md          # 이용약관
└── README.md         # 본 파일
```

## 문서 수정 시

1. `.md` 파일 편집
2. `git commit + push`
3. GitHub Pages가 자동으로 빌드 (1~2분)
4. URL 새로고침 시 변경사항 반영

## 앱 내 마크다운과 동기화

앱 내 표시용 마크다운 (`obd_app/legal/`)과 본 저장소를 **수동으로 동기화**해야 합니다.
변경 시 양쪽 모두 업데이트하세요.

## 시행일 / 개정 시 주의

처리방침 변경은 **이용자에게 미리 고지**해야 합니다.
- 일반 변경: 시행 7일 전 고지
- 중대 변경: 시행 30일 전 고지
